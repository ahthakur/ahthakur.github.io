## Cursor Cloud specific instructions

This is a minimal static HTML site (GitHub Pages). The repo contains a single `index.html` file.

- **No dependencies** to install — no package manager, no build tool, no linting, no tests.
- **Dev server**: serve locally with `python3 -m http.server 8080` from the repo root, then browse to `http://localhost:8080/`.
- There is no build step, lint step, or test suite.
